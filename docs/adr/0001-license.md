# ADR 0001: License = Apache-2.0

## Status

Accepted (2026-03-11)

## Context

`rikka-agent` is intended to be open source and easy for others (including companies) to use directly.

## Decision

We adopt the Apache License 2.0.

## Consequences

- Permissive use (including commercial use) is straightforward.
- Includes an explicit patent grant, which is commonly preferred by companies.
