# OSS / Community Spec

## 1) Repository Standards

- Apache-2.0 license
- Clear security reporting policy (`SECURITY.md`)
- Contribution guide (`CONTRIBUTING.md`)
- Threat model and server hardening guidance

## 2) Issue Templates (Later)

We may add:

- Bug report template
- Feature request template
- Security concern template (non-sensitive; points to SECURITY.md)

## 3) Supported Platforms

- Android only (initially)

## 4) Localization

- English-first for OSS reach, with optional Chinese docs where helpful
